# Cogu — Web App v3

Versão visual aprimorada do aplicativo web do Cogu.

## Melhorias da v3
- Visual mais sofisticado e mais “cara de jogo”.
- Tela inicial melhorada.
- Dashboard de rodada.
- Cartão de ação com destaque.
- Tela de derrota mais dramática.
- Preferências na tela inicial:
  - ligar/desligar som;
  - confirmar antes de puxar carta.
- Histórico mantido.
- Tela cheia.
- Instalação como app (quando o navegador permitir).

## Como usar
1. Extraia o ZIP.
2. Abra `index.html` no navegador.

### Melhor forma de testar
No terminal, dentro da pasta:
`python -m http.server 8000`

Depois abra:
`http://localhost:8000`

## Importante
Este pacote foi gerado com as cartas disponíveis no ambiente do projeto. Se você já substituiu pelas cartas finais no seu app local, pode simplesmente copiar os arquivos de interface desta v3 sobre sua pasta atual, ou trocar novamente a pasta:

`assets/cards/`
